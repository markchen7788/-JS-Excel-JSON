# JS简单实现读取Excel文件并转换为JSON

#### 介绍
1. 简单的使用了一下sheetJs,更多详情请看[官方仓库](https://github.com/SheetJS/sheetjs)
2. 可以实现简单的excel文件转Json;
3. 可以将相同格式的Json数组用 [Layui](https://www.layui.com) 数据表格显示。
